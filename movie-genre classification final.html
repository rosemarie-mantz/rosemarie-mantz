Εισαγωγή¶

Ο σκοπός της παρούσας εργασίας είναι η πρόβλεψη του είδους μιας ταινίας λαμβάνοντας υπόψη τη περιγραφή της πλοκής της. Το σύνολο των δεδομένων υπό εξέταση είναι το Wikipedia Movie Plots, το οποίο περιέχει γνωρίσματα από 34,886 ταινίες παγκοσμίως. Τα γνωρίσματα που έχουν εξαχθεί από κάθε ταινία είναι τα παρακάτω:

  • Release Year - Έτος πρεμιέρας της ταινίας
  • Title - Τίτλος της ταινίας
  • Origin/Ethnicity - Χώρα πρέλευσης ταινίας
  • Director - Σκηνοθέτης
  • Plot - Πρωταγωνιστές ταινίας (ηθοποιοί)
  • Genre - Είδος ταινίας
  • Wiki Page - URL της ιστοσελίδας Wikipedia από την οποία εξάχθηκαν τα δεδομένα
  • Plot - Περιγραφή της πλοκής της ταινίας

Για το ζητούμενο της εργασίας, απαιτούνται μόνο τα πεδία που αφορούν το είδος της ταινίας (Genre) και τη περιγραφή της πλοκής της ταινίας (Plot).

import re
import string
from bs4 import BeautifulSoup
from nltk.stem import PorterStemmer

# Create a Porter stemmer object
stemmer = PorterStemmer()

# Συνάρτηση για τον υπόλοιπο καθαρισμό του κειμένου
def preprocess_text(text):
    # Μετατροπή σε μικρά γράμματα
    text = text.lower()
    
    # Αφαίρεση (\'s)
    text = re.sub(r"\'s", "", text)
    
    # Αφαίρεση καινούριας γραμμής (\r\n)
    text = re.sub(r"\r\n", "", text)
    
    # Αφαίρεση του κειμένου που βρίσκεται μέσα σε παρένθεση ()
    text = re.sub(r"\([^()]*\)", "", text)

    # Αφαίρεση αριθμών
    text = re.sub(r"\d+", "", text)
    
    # Αφαίρεση σημείων στίξης και ειδικών χαρακτήρων
    text = text.translate(str.maketrans("", "", string.punctuation))
    
    # Αφαίρεση HTML ετικετών
    text = BeautifulSoup(text, "html.parser").get_text()
    
    # Αφαίρεση περιττών κενών
    text = re.sub(r"\s+", " ", text).strip()

    # Apply stemming
    stemmed_text = ' '.join([stemmer.stem(word) for word in text.split()])
    
    return stemmed_text

Μπορεί να παρατηρηθεί ότι, η μεταβλητή του είδους της ταινίας (genre) είναι κατηγορική μεταβλητή και ειδικότερα ονομαστική κατηγορική μεταβλητή (nominal categorical variable). Προκειμένου να μπορεί να αποτελέσει έγκυρη είσοδο, θα πρέπει να μετατραπεί σε αριθμητική μεταβλητή. Το τελευταίο, πραγματοποιείται μετατρέποντας τη σε Dummy Μεταβλητές (Dummy Variables) ή εναλλακτικά σε Κωδικοποίηση One-hot (One-hot Encoding). Ειδικότερα, αυτό που γίνεται είναι ότι δοθέντος μιας μεταβλητής με k επίπεδα, κάθε επίπεδο μετατρέπεται σε ένα νέο χαρακτηριστικό με την τιμή 1 (TRUE) εκεί που εμφανίζεται η τιμή του συγκεκριμένου επιπέδου και 0 (FALSE) αλλού. Η διαδικασία αυτή μπορεί να πραγματοποιηθεί μέσω της συνάρτησης get_dummies του pandas Dataframe. Τέλος, η μεταβλητή προς πρόβλεψη θα διατηρήθει σε μια ξεχωριστή μεταβλητή, ώστε να μπορεί να αξιοποιηθεί αργότερα.

Σε αυτό το σημείο, μπορεί να γίνει διαχωρισμός με τυχαία δειγματοληψία του συνόλου δεδομένων σε σύνολο δεδομένων εκπαίδευσης (train set) και σύνολο δεδομένων δοκιμής (test set), με αναλογία 70% και 30% του αρχικού συνόλου δεδομένων, αντίστοιχα. Έτσι, θα πραγματοποιηθεί εκπαίδευση των μεθόδων με τα δεδομένα εκπαίδευσης και στην συνέχεια θα διεκπεραιωθεί η αξιολόγηση τους στο σύνολο δεδομένων δοκιμής. Με αυτόν τον τρόπο, η μέθοδος αξιολογείται σύμφωνα με δεδομένα, που δεν έχει ξανα δει και συνεπώς η αποτίμηση της απόδοσης δεν εμπεριέχει προκαταληψη (bias). Για τον διαχωρισμό, που περιγράφηκε προηγουμένως, θα χρησιμοποιηθεί η συνάρτηση του module model_selection της βιβλιοθήκης scikit-learn με όνομα train_test_split.

Κατά την εκπαίδευση ενός RNN δικτύου, ενώ οι ακολουθίες εισόδου μπορεί να έχουν διαφορετικά μήκη, συνήθως υπόκεινται σε padding για να έχουν το ίδιο μήκος μέσα σε ένα batch ώστε να γίνει πιο αποτελεσματικός υπολογισμός. To padding περιλαμβάνει την προσθήκη μηδενικών, στη δική μας περίπτωση, στο τέλος των μικρότερων ακολουθιών, έτσι ώστε να ταιριάζουν με το μήκος της μεγαλύτερης ακολουθίας. Αυτό επιτρέπει στο RNN να επεξεργάζεται πολλές ακολουθίες ταυτόχρονα χρησιμοποιώντας λειτουργίες μήτρας, οι οποίες μπορούν να επιταχυνθούν από τις GPU. Το μεγαλύτερο μήκος πλοκής ταινίας ήταν 2804 λέξεις, που υπολογίστηκε παραπάνω. Άρα το padding ορίστηκε στις 3000 λέξεις, για να υπάρχει και περιθώριο.

Κατασκευή, Εκπαίδευση και Αξιολόγηση του Αναδρομικού Νευρωνικού Δικτύου¶

Σε αυτό το σημείο μπορεί να πραγματοποιηθεί η κατασκευή, εκπαίδευση και αξιολόγηση του Αναδρομικού Νευρωνικού Δικτύου. Συγκεκριμένα, για την επίλυση του προβλήματος της ταξινόμησης των ταινιών βάσει είδους, θα χρησιμοποιηθεί ένα Long Short-Term Memory (LSTM) τεχνητό νευρωνικό δίκτυο, το οποίο θα κατασκευαστεί μέσω του Functional API. Ειδικότερα, τα μοντέλα ορίζονται δημιουργώντας και συνδέοντας κατάλληλα τα επιθυμητά επίπεδα, ενώ στην συνέχεια γίνεται η δημιουργία του μοντέλου ορίζοντας από τα συνδεδεμένα επίπεδα ποιο θα αποτελεί την είσοδο και ποιό την έξοδο, καθώς ταυτόχρονα υποστηρίζει και πολλαπλές εισόδους και εξόδους. Η δημιουργία του μοντέλου γίνεται χρησιμοποιώντας την συνάρτηση Model που βρίσκεται στο module keras της βιβλιοθήκης Tensorflow.

import tensorflow as tf

# Δημιουργία του επιπέδου εισόδου
input_layer = tf.keras.Input(shape=(3000,), dtype="int32") # Είσοδος μεγέθους 3000

# Δημιουργία επιππέδου Embedding
x = tf.keras.layers.Embedding(input_dim=10000, # διαστάσεις εισόδου
                              output_dim=32, # διαστάσεις εξόδου
                              embeddings_regularizer=tf.keras.regularizers.l2(0.01))(input_layer) #regularization L2

# Δημιουργία LSTM τεχνητού νευρωνικού δικτύου
x = tf.keras.layers.LSTM(units=32, # χρησιμοποίησε 32 νευρώνες σε κάθε εσωτερικό κελί
                         return_sequences=True)(x)  # return_sequences=True

# Δημιουργία LSTM τεχνητού νευρωνικού δικτύου με recurrent dropout 
x = tf.keras.layers.LSTM(units=32, recurrent_dropout=0.5)(x)  # ρυθμός recurrent dropout 0.5

# Δημιουργία Επιπέδου ταξινόμησης
out = tf.keras.layers.Dense(units=num_classes, activation="softmax")(x)

# Δημουργία τελικού Μοντέλου
lstm_model = tf.keras.Model(inputs=input_layer, outputs=out)
Αρχικά, το Input Layer ορίζει το σχήμα και τον τύπο δεδομένων της εισόδου στο μοντέλο. Σε αυτήν την περίπτωση, το σχήμα εισόδου είναι (3000,), δηλαδή το μοντέλο αναμένει πλοκές ταινιών μήκους 3000 λέξεων. Αμέσως μετά, Embedding Layer μετατρέπει τις ακέραιες ακολουθίες εισόδου σε dense vector representations. Αντιστοιχίζει κάθε ακέραια τιμή σε ένα dense vector μεγέθους output_dim, σε αυτήν την περίπτωση, 32. Η παράμετρος input_dim καθορίζει το μέγεθος του λεξιλογίου και το embeddings_regularizer εφαρμόζει regularization L2 στα βάρη του embedding με ρυθμό 0,01, ώστε να αποφευχθεί το overfitting. Έπειτα προστίθεται ένα LSTM Layer. Η παράμετρος units καθορίζει τον αριθμό των κρυμμένων νευρώνων LSTM στο επίπεδο. Βάζοντας τη παράμετρο return_sequences=True σημαίνει ότι το επίπεδο θα επιστρέψει ολόκληρη την ακολουθία των κρυφών καταστάσεων και όχι μόνο την τελευταία. Αυτό είναι αναγκαίο διότι προστίθεται ένα ακόμα LSTM Layer στο μοντέλο, με τον ίδιο αριθμό μονάδων (32) με το προηγούμενο επίπεδο. Ωστόσο, αυτό το επίπεδο περιλαμβάνει επίσης reccurent dropout, η οποία βοηθά στην αποφυγή overfitting σε επαναλαμβανόμενα νευρωνικά δίκτυα θέτοντας τυχαία ένα κλάσμα των εισόδων στο 0 κατά τη διάρκεια της εκπαίδευσης. Η παράμετρος recurrent_dropout ελέγχει το ποσοστό dropout, το οποίο έχει οριστεί σε 0,5, που σημαίνει ότι το 50% των input θα απορριφθεί κατά τη διάρκεια της εκπαίδευσης. Τέλος, το Dense Layer είναι ένα πλήρως συνδεδεμένο επίπεδο που αντιστοιχίζει την έξοδο του προηγούμενου επιπέδου με τον αριθμό των κλάσεων εξόδου (num_classes). Η συνάρτηση ενεργοποίησης έχει οριστεί σε "softmax", η οποία παράγει μια κατανομή πιθανότητας στις κλάσεις, υποδεικνύοντας την πιθανότητα κάθε κλάσης.

Στο σημείο αυτό, είναι εφικτή η έναρξη της εκπαίδευσης του κατασκευασμένου Αναδρομικού Νευρωνικού Δικτύου. Συγκεκριμένα, η διαδικασία αυτή θα διεκπεραιωθεί μέσω της μεθόδου fit του μοντέλου που δημιουργήθηκε. Στα δύο πρώτα ορίσματα τοποθετούνται τα δεδομένα εκπαίδευσης, δηλαδή οι περιγραφές της πλοκής της ταινίας και τα αντίστοιχα είδη τους που εμπεριέχονται στο σύνολο δεδομένων εκπαίδευσης, ενώ ταυτόχρονα η εκπαίδευση θα πραγματοποιηθεί για 10 εποχές με μέγεθος της ομάδας παρατηρήσεων (batch size) ανα βήμα της Στοχαστικής Κατάβασης Παραγώγου (Stochastic Gradient Descent - SGD) ίσο με 64. Το μέγεθος της ομάδας παρατηρήσεων (batch size), σημαίνει ότι σε κάθε επανάληψη, ο αλγόριθμος βελτιστοποίησης επιλέγει τυχαία μια ομάδα παρατηρήσεων πληθικότητας ίση με την τιμή της παραμέτρου, για τον υπολογισμό της παραγώγου (gradient). Σε αυτήν την περίπτωση θα χρησιμοποιηθεί μέγεθος ομάδας παρατηρήσεων ίσο με 64. Επίσης, η παράμετρος epochs, αναφέρεται στο πλήθος επόχων (epochs), όπου μία εποχή αντιστοιχεί σε ένα πέρασμα όλων των δεδομένων εκπαίδευσης από το νευρωνικό δίκτυο. Εναλλακτικά, μία εποχή είνα ίση με το πλήθος των επαναλήψεων του αλγορίθμου βελτιστοποίησης, που απαιτούνται για την επεξεργασία όλων των παρατηρήσεων του συνόλου δεδομένων εκπαίδευσης. Στο παράδειγμα αυτό, θα χρησιμοποιηθούν 10 εποχές. Τέλος, παρέχεται σύνολο δεδομένων επαλήθευσης μέσω της παραμέτρου validation_split και ορίζεται ίσο με το 10% του συνόλου δεδομένων εκπαίδευσης. Η παράμετρος validation_data, επιτρέπει την εισαγωγή δεδομένων επαλήθευσης, δηλαδή δεδομένων, που δεν χρησιμοποιούνται στην εκπαίδευση του δικτύου, αλλα μας επιτρέπουν να παρακολουθούμε την πρόοδου του με το πέρασμα των εποχών. Στο παράδειγμα αυτό, χρησιμοποιούνται τα δεδομένα δοκιμής, ώστε να μπορεί να παρατηρηθεί η τιμή του Μέσου Απόλυτου Σφάλματος τόσο των δεδομένων εκπαίδευσης, όσο και των δεδομένων δοκιμής κατά το πέρασμα των εποχών.
